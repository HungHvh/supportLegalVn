## 🏗️ Architecture

The `supportLegalVn` architecture is built on a **Modular RAG Pipeline** with three primary stages:

1.  **Ingestion & Parsing**:
    *   **Recursive Legal Parser**: A custom parser that respects the hierarchy of Vietnamese legal documents (Phần, Chương, Mục, Điều).
    *   **Hierarchical Chunking**: Metadata inheritance ensures that child chunks (paragraphs) retain the context of their parent documents (Titles, Chapters).
2.  **Hybrid Retrieval Engine**:
    *   **Dense Search**: Qdrant vector database using `vietnamese-sbert`.
    *   **Sparse Search**: SQLite FTS5 for exact keyword matching.
    *   **RRF Fusion**: Reciprocal Rank Fusion combines both result sets for optimal relevance.
3.  **Intelligent Generation**:
    *   **Query Classification**: A pre-query LLM classifier (Groq/DeepSeek) detects the legal domain to apply metadata filters, reducing search noise.
    *   **Provider-Agnostic Generation**: A flexible generator support streaming responses from Groq (Llama-3.3-70b), Gemini, or DeepSeek.

## ⚖️ Key Decisions

*   **Custom Legal Parser (Phase 5)**: Decided against generic markdown chunkers in favor of a recursive parser that mirrors the structure of Vietnamese law. This significantly improved the quality of "Rule" (R) in IRAC responses.
*   **Provider Failover (Phase 6.2-6.4)**: Implemented a provider-agnostic client wrapper. If Groq/DeepSeek hits rate limits, the system can transparently fail back to Gemini, ensuring high availability.
*   **Domain-Aware Filtering**: Chose to classify queries *before* retrieval. This ensures that a query about "Criminal Law" never pulls snippets from "Civil Law" documents, even if keywords overlap.

## 🚀 Getting Started (for New Team Members)

1.  **Setup**: Copy `.env.example` to `.env` and add `GROQ_API_KEY`, `DEEPSEEK_API_KEY`, and `GEMINI_API_KEY`.
2.  **Infrastructure**: Run `docker-compose up -d` to start Qdrant and the API.
3.  **Indexing**: To populate the DB, run `python indexer.py --limit 100` (starts streaming from HF).
4.  **Querying**: Use the `/ask` endpoint or run `scripts/smoke_test_rag.py`.
